# typestart-koa
## Typescript + Koa JumpStart Project


```bash
git clone git@github.com:jcirizar/typestart-koa.git
cd typestart-koa
npm install
```

## Develop

### Run server and reload on changes
`npm run watch:server`
### Run tests and rerun on changes
`npm run watch:test`
